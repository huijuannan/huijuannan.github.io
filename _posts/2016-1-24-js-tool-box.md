## npm

```
npm install
```

## [bower](http://bower.io/#getting-started)

前端库管理工具。比如bootsrap, jQuery等.

- 安装`bower`
```
npm install -g bower
```
- 安装jquery
```
bower install jquery
```
- 创建bower.json文档
```
bower init
```
- 向bower.json里添加dependencies
```
bower install Package --save
```
